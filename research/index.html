    Cognitive Robotics + Dialog Systems

    • Development of a Memory Storage System for the robot Indy at RIKEN Guardian Robot Project. This memory system is built using concepts from Cognitive Psychology, particularly research on memory decay or "forgetting", along with systems that store, adjust, and summarize past experiences in natural language with the aid of Large-Language Models.
    • Development of a Dialogue System that incorporates the robot's stored memories by retrieving past experiences that match the current dialogue context and incorporate it in the robot's spoken responses.
    • Technical support for Automatic Speech Recognition and other modules.
